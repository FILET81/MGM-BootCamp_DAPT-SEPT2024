Neural Network Hyperparameter Tuning

In this section, we delve into an investigation of hyperparameters to enhance the efficiency of a neural network. The hyperparameters under consideration include the number of training epochs, batch size, hidden layer count, and the quantity of neurons within each layer. We specifically limit the exploration of layer counts to 1, 2, and 3, while varying the number of neurons per layer across 5, 10, and 15. The rationale behind constraining the layer and neuron count lies in recognizing the inherent simplicity of the input features. As most of the input features are binary variables derived from categorical variables through one-hot encoding, a sophisticated neural network architecture is deemed unnecessary. Consequently, our focus remains on a selection of modest architectures to strike a balance between model expressiveness and computational efficiency.

Utilizing a grid search methodology, we systematically assess the neural network's performance across diverse combinations of hyperparameter values. This approach allows us to pinpoint the hyperparameter set that demonstrates optimal performance on the validation sets within a k-fold cross-validation framework.

Our evaluation of hyperparameter sets revolves around two performance metrics: accuracy and the area under the receiver operating characteristic curve (AUC). These metrics provide a comprehensive insight into the model's proficiency in accurately classifying observations, within both majority and minority classes.

Following the completion of the grid search, our primary aim is to identify the hyperparameter set exhibiting the highest accuracy and AUC on the validation sets within the k-fold cross-validation. This best-performing hyperparameter set will subsequently serve as the base for the final model configuration, embodying a compromise between computational efficiency and predictive accuracy.

# Function to define the nn model
def define_model(input_dim, layers, neurons, loss_f, metric, learning_rate):
    model = Sequential()
    model.add(Dense(neurons, input_shape=(input_dim,), activation='relu', kernel_initializer='he_uniform'))
    
    for _ in range(layers - 1):
        model.add(Dense(neurons, activation='relu', kernel_initializer='he_uniform'))
    
    model.add(Dense(1, activation='sigmoid'))  # Output - binary
    opt = SGD(learning_rate=learning_rate, momentum=0.9)
    model.compile(optimizer=opt, loss=loss_f, weighted_metrics=metric)
    return model

# Function to evaluate the model using k-fold cross-validation
def evaluate_model(X_train, y_train, n_folds, loss_f, metric, epochs, batch_size, layers, neurons, learning_rate):
    scores, histories, aucs = list(), list(), list()
    kfold = KFold(n_folds, shuffle=True, random_state=1)
    
    for train_ix, test_ix in kfold.split(X_train):
        trainX, trainY, testX, testY = X_train.iloc[train_ix, :], y_train.iloc[train_ix], X_train.iloc[test_ix, :], y_train.iloc[test_ix]
        
        # Add Early stop
        early_stop = EarlyStopping(monitor='val_loss', patience=5, restore_best_weights=True)
        
        model = define_model(X_train.shape[1], layers, neurons, loss_f, metric, learning_rate)
        
        # Fit model
        history = model.fit(trainX, trainY, epochs=epochs, batch_size=batch_size,
                            validation_data=(testX, testY), verbose=0, callbacks=[early_stop])
        
        # Evaluate model
        loss, acc, auc = model.evaluate(testX, testY, verbose=0)
        
        # Compute AUC
        y_pred_proba = model.predict(testX, verbose=0)
        auc = roc_auc_score(testY, y_pred_proba)
        
        print('> Accuracy: %.3f, AUC: %.3f' % (acc * 100.0, auc * 100.0))
        
        
        # Append scores
        scores.append(acc)
        aucs.append(auc)
        histories.append(history)
        
    return scores, aucs, histories, model

# Function to summarize model performance
def summarize_performance(scores, aucs):
    print('AUC: mean=%.3f std=%.3f, n=%d' % (np.mean(aucs)*100, np.std(aucs)*100, len(aucs)))

    print('Accuracy: mean=%.3f std=%.3f, n=%d' % (np.mean(scores)*100, np.std(scores)*100, len(scores)))
    
    plt.boxplot(scores)
    plt.title('Validation Accuracy Scores of 5-Fold')
    plt.show()

# Function to run the test harness for evaluating a model
def run_test_harness(X_train, y_train, n_folds, loss_f, metric, epochs, batch_size, layers, neurons, learning_rate):
    scores, aucs, histories, model = evaluate_model(X_train, y_train, n_folds, loss_f, metric, 
                                                    epochs, batch_size, layers, neurons, learning_rate)
#     summarize_diagnostics(histories)
    summarize_performance(scores, aucs)
    return scores, aucs
